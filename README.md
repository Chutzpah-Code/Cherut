## About

Cherut is a web application that helps you organize and track every dimension of your life — from high-level goals and OKRs down to daily tasks, habits, time tracking, and journaling.

## Getting Started

### Requirements

- Node.js 18+
- Firebase project (Auth + Firestore)

### Running locally

```bash
# Backend
cd apps/api
npm install
npm run start:dev

# Frontend
cd apps/web
npm install
npm run dev
```

- Web: http://localhost:3000
- API: http://localhost:4000

For emulator mode, see `CLAUDE.md` → Firebase Emulator Mode.
